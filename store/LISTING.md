## Long Description

**Export all your Looker Studio chart data in one click.**

Looker Studio (formerly Google Data Studio) doesn't let you bulk-export chart data. This extension fixes that.

**What it does:**
- Captures data from every chart as the report loads — tables, bar charts, pie charts, scorecards, and more
- Shows a list of captured charts in the popup with row and column counts
- Exports selected charts as individual CSV files, or all at once as a ZIP
- Supports multi-page reports: click "All Pages" to automatically navigate each page and capture everything
- Fetches all rows for any chart, bypassing the default display limit (great for bar charts that show only top 10)

**How to use:**
1. Open any Looker Studio report you have access to
2. Click the extension icon — charts appear as data loads
3. Select the charts you want and click Export CSV
4. For multi-page reports, click "All Pages" to capture every page automatically

**Works with:**
- lookerstudio.google.com
- datastudio.google.com (legacy URLs)
- Reports connected to Google Analytics 4, BigQuery, Google Sheets, and other data sources

**Privacy:**
This extension does not collect, transmit, or store any data outside your browser. All data stays on your machine. See the Privacy Policy for full details.
